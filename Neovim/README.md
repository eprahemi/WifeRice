# Neovim Configuration

> **Install path:** `~/.config/nvim/`

Neovim text editor configuration.

| File | Purpose |
|------|---------|
| `init.lua` | Main Neovim config (plugins, keybindings, settings) |
| `lazy-lock.json` | Locked plugin versions for reproducible installs |
| `matugen_colors.lua` | Dynamic color scheme generated by Matugen |
