# Hyprland Configuration

Part of [Eprahemi](https://github.com/eprahemi/WifeRice)

My custom Hyprland setup with QuickShell widgets, Matugen theming, and dynamic wallpaper integration.

## Features
- Custom QuickShell status bar and widgets
- Dynamic theming via Matugen (changes colors based on wallpaper)
- Update notification system linked to `eprahemi/WifeRice`
- Built-in wallpaper changer and lockscreen
- Polished animations and effects

## Install Location
`~/.config/hypr/`
