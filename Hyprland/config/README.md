# Hyprland Modular Configs

> **Install path:** `~/.config/hypr/config/`

Individual configuration modules sourced by `hyprland.conf`.

| File | Purpose |
|------|---------|
| `autostart.conf` | Startup applications (QuickShell, daemons, agents) |
| `env.conf` | Environment variables (XCURSOR, GDK, etc.) |
| `keybindings.conf` | Keyboard shortcuts, media keys, workspace navigation |
| `monitors.conf` | Monitor resolution, position, refresh rate |
| `rules.conf` | Window rules (workspace assignment, floating, sizing) |
| `settings.conf` | Auto-generated by settings watcher script |
| `variables.conf` | Global variable definitions ($mainMod, $terminal, $browser, etc.) |
