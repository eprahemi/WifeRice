# Matugen Templates

> **Install path:** `~/.config/hypr/templates/`

Configuration templates processed by the Matugen theme engine.
When the wallpaper changes, Matugen extracts dominant colors and injects them into these templates, generating the final config files.

| Template | Generates |
|----------|-----------|
| `autostart.conf.template` | `~/.config/hypr/config/autostart.conf` |
| `env.conf.template` | `~/.config/hypr/config/env.conf` |
| `keybinds.conf.template` | `~/.config/hypr/config/keybindings.conf` |
| `monitors.conf.template` | `~/.config/hypr/config/monitors.conf` |
| `settings.conf.template` | `~/.config/hypr/config/settings.conf` |
