# Matugen Theme Engine

> **Install path:** `~/.config/matugen/`

Matugen extracts dominant colors from wallpapers and dynamically applies them across all applications.

## Main Config

| File | Purpose |
|------|---------|
| `config.toml` | Matugen settings (color scheme, contrast, hooks) |

## Templates

> **Install path:** `~/.config/matugen/templates/`

Templates are processed by Matugen to generate color-themed config files.

| Template | Generates |
|----------|-----------|
| `cava-colors.ini.template` | Cava audio visualizer colors |
| `gtk.css.template` | GTK theme colors |
| `hyprland.conf.template` | Hyprland color variables |
| `kitty-colors.conf.template` | Kitty terminal colors |
| `nvim-colors.lua.template` | Neovim color scheme |
| `qs_colors.json.template` | QuickShell widget colors |
| `qtct.conf.template` | Qt configuration tool colors |
| `qt-style.qss.template` | Qt stylesheet |
| `rofi.rasi.template` | Rofi launcher colors |
| `sddm-colors.qml.template` | SDDM login screen colors |
| `swaync.css.template` | Notification center colors |
| `swayosd.css.template` | OSD volume/brightness indicator colors |
